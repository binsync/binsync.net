Git Based

Sync in Real Time

Changes sync automatically across all connected decompilers in real-time.

No Tool Boundaries

Seamlessly work across multiple decompilers without workflow interruption.

Scriptable

BinSync, and its libraries, are built to run on most decompilers on Windows, MacOS, and Linux.
